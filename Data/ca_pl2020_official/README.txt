2020 Redistricting Data for California at the Block level (Adjusted)

##Redistricting Data Hub (RDH) Retrieval Date
09/28/2021

##Sources
This dataset was retrieved from the Statewide Database at https://statewidedatabase.org/redistricting2021/counties.html#

##Fields
Contained in this zip folder are the adjusted P2 (with DOJ racial/ethnic cateogries), P4 (with DOJ racial/ethnic categories), and P5. 
Fields and additional metadata information such as field headers, data dictionaries, and dataset time stamp updates are included for each dataset.

#Processing
The RDH retrieved the data from the source listed and did not modify any of the data.
